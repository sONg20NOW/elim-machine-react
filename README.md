# 엘림 기계설비현장 웹 애플리케이션
[엘림 주식회사 내사용 기계설비현장 관리 웹애플리케이션](https://elim.hs9024.cloud)입니다.
- 현장 정보 관리
- 설비 인력 관리
- 직원 정보 관리
- 라이선스 정보 관리
- 보고서 다운로드
